Follow firebase document: https://firebase.google.com/docs/flutter/setup?platform=ios

Suggestion:
    - Install Firebase CLI by using npm for window. 
        Requirement Node.js v18.0.0 or later. Check Node version by "node --version".
        > npm install -g firebase-tools
    - After run "dart pub global activate flutterfire_cli"
        you may need to set path environtment. just copy path from their message.
    
    - After setup Firebase, The build will failed.
        The message look like below.

            FAILURE: Build failed with an exception.

            * What went wrong:
            Execution failed for task ':app:processDebugMainManifest'.
            > Manifest merger failed : uses-sdk:minSdkVersion 21 cannot be smaller than version 23 declared in library [:firebase_core] 

            ...

            ┌─ Flutter Fix ─────────────────────────────────────────────────────────────────────────────────┐
            │ The plugin firebase_core requires a higher Android SDK version.                               │
            │ Fix this issue by adding the following to the file                                            │
            │ C:\Work\Flutter-Spotify\android\app\build.gradle:                                             │
            │ android {                                                                                     │
            │   defaultConfig {                                                                             │
            │     minSdkVersion 23                                                                          │
            │   }                                                                                           │
            │ }                                                                                             │
            │                                                                                               │
            │ Following this change, your app will not be available to users running Android SDKs below 23. │
            │ Consider searching for a version of this plugin that supports these lower versions of the     │
            │ Android SDK instead.                                                                          │
            │ For more information, see: https://flutter.dev/to/review-gradle-config                        │
            └───────────────────────────────────────────────────────────────────────────────────────────────┘
    
        Solution
            Ref: https://stackoverflow.com/a/78241257/29800584

            1. Open file "FlutterExtension.kt" localed at : C:\{your_flutter_folder}\flutter\packages\flutter_tools\gradle\src\main\kotlin

            2. Change minSdkVersion to 23.
                val minSdkVersion: Int = 23
            
            3. Run Flutter as your normal way.
